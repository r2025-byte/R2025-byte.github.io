Planning & Scheduling (Primavera)

This course equips participants with essential skills to plan, schedule, and control projects using Primavera P6. It focuses on creating realistic project schedules, managing resources, tracking performance, and applying Earned Value Management techniques for effective project control.

Course Objectives

  • Understand the fundamentals of project planning and scheduling.
  • Develop and manage project baselines using Primavera P6.
  • Allocate and control project resources efficiently.
  • Apply earned value analysis and performance measurement techniques.

Topics Covered

  • Introduction to Project Planning Concepts
  • Work Breakdown Structure (WBS) Development
  • Activity Definition and Sequencing
  • Scheduling Techniques (Critical Path Method)
  • Resource Allocation and Leveling
  • Tracking Progress and Updating Schedules
  • Earned Value Management

Course Details

  • Duration: 24 Hours
  • Level: Intermediate to Advanced
  • Language: English
  • Instructor: Eng. Moh’d Kamal
